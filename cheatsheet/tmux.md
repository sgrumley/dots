# TMUX Key Bindings
## Windows (tabs)
`c` - create window 
`w` - list windows 
`n` - next window 
`p` - previous window 
`f` - find window 
`,` - name window 
`&` - kill window
`1-9` - select window

## Panes (splits)
`%` - vertical split 
`"` - horizontal split 
`o` - swap panes 
`{` - move current pane left
`}` - move current pane right
`z` - toggle pane zoom

`-` - break pane into window
`*` - restore pane from window

## Copy Mode
`[` - enter copy mode
`v` - start selection
`y` - copy selection

## General 
`I` - source conf
